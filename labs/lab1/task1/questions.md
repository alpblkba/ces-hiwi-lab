# Lab 1.1 questions for self-check

These questions are used to check whether the environment bring-up was understood and documented correctly.

## Environment questions

1. Which machine are you working on, and how did you verify it?
2. What is your current working directory?
3. Is the working directory writable?
4. Which shell are you using?
5. Why is it important to run tool commands from a writable working directory?

## Xilinx toolchain questions

1. Which Xilinx version was selected for this lab?
2. How did you load the selected Xilinx environment?
3. Which executable path was resolved for Vivado?
4. Which executable path was resolved for Vitis HLS?
5. Which executable path was resolved for Vitis?
6. Why should Vivado, Vitis HLS, and Vitis use the same selected lab version?
7. What can go wrong if a generated project is created with the wrong tool version?
