# CES HiWi Lab Work

Personal working repository for CES HiWi lab preparation and development.

## Scope

- server setup notes
- Xilinx/Vivado/Vitis HLS environment checks
- basic digital design warm-up
- HLS learning notes
- small reproducible experiments
- lab-related helper scripts

## Baseline environment

Current baseline Xilinx version: 2022.2.

## Repository structure

- `notes/`: learning notes and setup notes
- `env/`: environment helper scripts
- `scripts/`: small utility scripts
- `designs/`: RTL/digital-design warm-up examples
- `hls/`: Vitis HLS examples and notes
- `experiments/`: throwaway experiments
- `docs/`: longer documentation drafts
